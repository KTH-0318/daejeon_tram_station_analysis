# daejeon_tram_station_analysis

"역 위치가 생활권과 맞지 않아 이용률이 낮다"는 가설에서 출발해,
**네트워크 중심성(서울 2호선 벤치마킹) → KMeans 군집 → 시나리오 분석 → TOPSIS → GIS 접근성 등급**을
하나의 의사결정 파이프라인으로 엮어 정거장 우선순위와 **신규 정거장 후보 입지를 제안**한 공간 분석 프로젝트.
